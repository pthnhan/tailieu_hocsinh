$\displaystyle{\text{A}} = \frac{x}{{x - 4}} - \frac{1}{{2 - \sqrt x }} + \frac{1}{{2 + \sqrt x }}\text{\quad và \quad}{\text{B}} = \frac{{\sqrt x  + 3}}{{\sqrt x  - 2}}\text{\;với\;}x \geqslant 0;x \ne 4$
\begin{enumerate}
\item Tính giá trị biểu thức B khi $x =25$
\item Rút gọn biểu thức A
\item Tìm những giá trị nguyên để biểu thức C = B : A nguyên
\end{enumerate}
\loigiai{
\begin{enumerate}
\item Khi $x=25$ thì $B = \dfrac{\sqrt {25}  + 3}{\sqrt {25}  - 2} = \dfrac{5 + 3}{5 - 2} = \dfrac{8}{5}$ 
\item  
\begin{align*}
  A &= \frac{x}{{x - 4}} - \frac{1}{{2 - \sqrt x }} + \dfrac{1}{2 + \sqrt x }  \\
   &= \frac{x}{{x - 4}} - \frac{{2 + \sqrt x }}{{4 - x}} + \frac{{\sqrt x  - 2}}{{x - 4}}  \\
   &= \frac{x}{{x - 4}} + \frac{{2 + \sqrt x }}{{x - 4}} + \frac{{\sqrt x  - 2}}{{x - 4}}  \\
   &= \frac{{x + 2\sqrt x }}{{x - 4}}  \\
   &=\frac{{\sqrt x }}{{\sqrt x  - 2}}\\
\end{align*}
\item 
\begin{align*}
  C = \frac{B}{A}& = \frac{\sqrt x  + 3}{\sqrt x  - 2}:\frac{\sqrt x }{\sqrt x  - 2}  \\
   &= \frac{\sqrt x  + 3}{\sqrt x  - 2}.\frac{\sqrt x  - 2}{\sqrt x } \\
   &= \frac{\sqrt x  + 3}{\sqrt x }  \\ 
   &= 1 + \frac{3}{\sqrt x }\\
\end{align*} 
Để C nguyên thì $3$ phải chia hết cho$\sqrt x$ hay $\sqrt x$ là ước của $3$. Các ước khác 0 của $3$ là $\left\{ { \pm 1; \pm 3} \right\}$\\
Những do $x > 0$, nên $x$ chỉ nhận các giá trị là $x = 1; 3$.
\end{enumerate}
}
