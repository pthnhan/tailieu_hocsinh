Cho biểu thức: $A = \dfrac{\sqrt x }{1 + 3\sqrt x },$ $B = \dfrac{x + 3}{x - 9} + \dfrac{2}{\sqrt x  + 3} - \dfrac{1}{3 - \sqrt x }$ với $x > 0,x \ne 9$.
\begin{enumerate}
\item Tính giá trị biểu thức $A$ khi $x = \dfrac{4}{9}$.
\item Rút gọn $B$.
\item Cho $P = \dfrac{B}{A}$, tìm $x$  để $P<3$.
\end{enumerate}
\loigiai{
\begin{enumerate}
\item Khi $x = \dfrac{4}{9}$ thì $\sqrt x  = \dfrac{2}{3}$ thì $A = \dfrac{{\dfrac{2}{3}}}{{1 + 3\cdot\dfrac{2}{3}}} = \dfrac{2}{9}$.
\item Ta có
\begin{align*}
B &= \dfrac{x + 3}{x - 9} + \dfrac{2}{\sqrt x  + 3} - \dfrac{1}{3 - \sqrt x }\\
 &= \dfrac{x + 3}{x - 9} + \dfrac{2(\sqrt x  - 3)}{x - 9} + \dfrac{\sqrt x  + 3}{x - 9}\\
 &= \dfrac{{\sqrt x \left( {\sqrt x  + 3} \right)}}{{\left( {\sqrt x  - 3} \right)\left( {\sqrt x  + 3} \right)}}\\
 &= \dfrac{\sqrt x }{\sqrt x  - 3}.
\end{align*}
\item Ta có

$$P = \dfrac{B}{A}=\dfrac{\sqrt x }{\sqrt x  - 3}:\dfrac{\sqrt x }{1 + 3\sqrt x } =\dfrac{3\sqrt x  + 1}{\sqrt x  - 3}$$
Vậy:
\begin{align*}
P < 3 &\Leftrightarrow \dfrac{3\sqrt x  + 1}{\sqrt x  - 3} < 3\\
&\Leftrightarrow \dfrac{3\sqrt x  + 1}{\sqrt x  - 3} - \dfrac{3\left( {\sqrt x  - 3} \right)}{\sqrt x  - 3} < 0\\
&\Leftrightarrow \dfrac{10}{\sqrt x  - 3} < 0\\
&\Leftrightarrow \sqrt x  - 3 < 0\\
&\Leftrightarrow x < 9.
\end{align*}
Kết hợp với điều kiện ban đầu ta suy ra $0 < x < 9$ thì $P < 3$.
\end{enumerate}
}
\end{ex}

\begin{ex}%[Đề thi Toán 9 Học kỳ 2 năm học 2017-2018, Quận Cầu Giấy, Hà Nội]%[9D3K4]
\textbf{\textit{Giải bài toán bằng cách lập phương trình hoặc hệ phương trình}}\\
Hai công nhân cùng làm chung một công việc thì trong $8$ giờ sẽ hoàn thành. Nếu mỗi người làm một mình, để hoàn thành công việc người thứ nhất cần nhiều hơn người thứ $2$ là $12$ giờ. Hỏi nếu làm riêng, thì mỗi người sẽ hoàn thành công việc trong bao lâu?
\loigiai{
Giả sử người 1 làm riêng trong $x$  (giờ)  thì hoàn thành công việc (điều kiện: $x > 12)$.\\
Giả sử người 2 làm riêng trong $y$  (giờ)  thì hoàn thành công việc (điều kiện: $y > 8,y < x$).\\
Trong một giờ người 1 làm được: $\dfrac{1}{x}$ công việc, người 2 làm được $\dfrac{1}{y}$  công việc.\\
Theo giả thiết hai người làm chung thì hoàn thành công việc trong 8 giờ nên ta có phương trình:
\begin{align*}
\dfrac{1}{x} + \dfrac{1}{y} = \dfrac{1}{8}\tag{1}
\end{align*}
Khi làm riêng người thứ nhất cần nhiều hơn người thứ 2 là 12 giờ để hoàn thành công việc nên ta có phương trình:
\begin{align*}
x - y = 12	\tag{2}
\end{align*}
Từ (1) và (2) ta có hệ phương trình: \\
\begin{eqnarray*}
&&\heva{&\dfrac{1}{x} + \dfrac{1}{y} = \dfrac{1}{8}\\&x - y = 12} \\
&\Leftrightarrow& \heva{&8\left( {x + y} \right) = xy\\&x = y + 12}\\
&\Leftrightarrow&\heva{&8\left( {2y + 12} \right) = y\left( {y + 12} \right)\\&x = y + 12}\\
&\Leftrightarrow&\heva{&y^2 - 4y - 96 = 0&(*)\\&x = y + 12.}
\end{eqnarray*}
$(*)\Leftrightarrow \left(y-12\right)\left(y+8\right) = 0 \Leftrightarrow \hoac{&y=12 \textrm { (thỏa mãn điều kiện)}\\&y=-8 \textrm{ (không thỏa mãn điều kiện).}}$\\  
Vậy nếu làm riêng người thứ 1 cần 12  giờ để hoàn thành công việc, người thứ 2 cần $12 + 12 = 24$ giờ để hoàn thành công việc.
}
\end{ex}

\begin{enumerate}
\item Giải hệ phương trình: $\heva{&\dfrac{1}{2x - 1} + \dfrac{4}{y + 5} = 3\\&\dfrac{3}{2x - 1} - \dfrac{2}{y + 5} =  - 5.}$ 
\item	Cho phương trình: $x^2 - 2(m + 1)x + 2m = 0$.\\
Chứng minh rằng phương trình đã cho luôn có 2 nghiệm phân biệt $x_1,x_2$ với mọi $m$.\\
Tìm $m$  để 2 nghiệm $x_1,x_2$  là độ dài 2 cạnh góc vuông của tam giác vuông có độ dài cạnh huyền bằng $\sqrt {12} $.\\
\end{enumerate}
\loigiai{
\begin{enumerate}
\item Xét hệ:
$\heva{&\dfrac{1}{2x - 1} + \dfrac{4}{y + 5} = 3\\
&\dfrac{3}{2x - 1} - \dfrac{2}{y + 5} =  - 5}$ $(I)$\\
Điều kiện: $\heva{&x \ne \dfrac{1}{2}\\&y \ne  - 5}$.\\
Khi đó, $\left( I \right) \Leftrightarrow \left\{ \begin{array}{l}
\dfrac{1}{2x - 1} + \dfrac{4}{y + 5} = 3\\
\dfrac{6}{2x - 1} - \dfrac{4}{y + 5} =  - 10
\end{array} \right.\,\, \Leftrightarrow \left\{ \begin{array}{l}
\dfrac{1}{2x - 1} + \dfrac{4}{y + 5} = 3\\
\dfrac{7}{2x - 1} =  - 7
\end{array} \right. \Rightarrow \left\{ \begin{array}{l}
x = 0\\
y =  - 4.
\end{array} \right.$\\
So với điều kiện, thỏa mãn.\\
Vậy hệ phương trình có nghiệm $(x;y)=(0;-4)$ .


\item Xét  phương trình: $x^2 - 2(m+1)x + 2m = 0$.\\
Ta có $\Delta ' = (m+1)^2-2m=m^2+1$\\
Vì $m^2 \ge 0, \forall m \Rightarrow \Delta' \ge 1 > 0, \forall m$ nên phương trình luôn có 2 nghiệm phân biệt $x_1, x_2$ .\\
Để  2 nghiệm $x_1,x_2$  là độ dài 2 cạnh góc vuông của tam giác vuông có độ dài cạnh huyền bằng $\sqrt {12} $ thì $x_1,x_2$ là các số thực dương thỏa mãn $x_1^2 + x_2^2 = 12$.\\ 
Theo hệ thức Viet ta có:
\begin{eqnarray*}
&&\left\{ \begin{array}{l}
x_1 + x_2 = 2(m+1)\\
x_1x_2 = 2m
\end{array} \right..
\end{eqnarray*}
Để $x_1,x_2 > 0$ $ \Rightarrow \left\{ \begin{array}{l}
2\left( {m + 1} \right) > 0\\
2m > 0
\end{array} \right. \Leftrightarrow m > 0$.\\
Ta có: $x_1^2 + x_2^2 = 12 \Leftrightarrow (x_1 + x_2)^2 - 2x_1x_2 = 12$\\
$ \Rightarrow 4(m+1)^2 - 4m - 12 = 0 \Leftrightarrow m^2 + m - 2 = 0$\\
$ \Leftrightarrow (m-1)(m+2) = 0$\\
$ \Leftrightarrow \left[ \begin{array}{l}
m = 1\\
m =  - 2
\end{array} \right.$.\\
Đối chiếu với điều kiện ta thấy $m = 1$  thỏa mãn.

\end{enumerate}


}
\end{ex}

Giải phương trình: $\sqrt {5x^2 + 4x}  - \sqrt {x^2 - 3x - 18}  = 5\sqrt x $ .
\loigiai{
Điều kiện: $\left\{ \begin{array}{l}
x^2 - 3x - 18 \ge 0\\
x \ge 0\\
5{x^2} + 4x \ge 0
\end{array} \right. \Leftrightarrow x \ge 6$.\\
\begin{align*}
&\sqrt {5x^2 + 4x}  - \sqrt {x^2 - 3x - 18}  = 5\sqrt x \\
\Leftrightarrow &\sqrt {5x^2 + 4x}  = \sqrt {x^2 - 3x - 18}  + 5\sqrt x\\
\Rightarrow &2x^2 - 9x + 9 - 5\sqrt {x(x^2 - 3x - 18)}  = 0\\
\Leftrightarrow &2(x^2 - 6x) + 3(x + 3) - 5\sqrt {(x^2 - 6x)(x + 3)}  = 0\\
\Leftrightarrow &2\cdot \dfrac{x^2 - 6x}{x + 3} - 5\sqrt {\dfrac{x^2 - 6x}{x + 3}}  + 3 = 0 (\textrm{vì } x + 3 > 0)
\end{align*}
Đặt $t = \sqrt {\dfrac{x^2 - 6x}{x + 3}}  \ge 0 \Rightarrow 2{t^2} - 5t + 3 = 0 \Leftrightarrow \left[ \begin{array}{l}
t = 1\\
t = \dfrac{3}{2}
\end{array} \right.$\\
TH1: $t = 1 \Leftrightarrow \sqrt {\dfrac{x^2 - 6x}{x + 3}}  = 1 \Rightarrow {x^2} - 7x - 3 = 0 \Leftrightarrow \left[ \begin{array}{l}
x = \dfrac{7 + \sqrt {61} }{2}\,\,{\textrm{(thoả mãn)}}\\
x = \dfrac{7 - \sqrt {61} }{2}\,\,\,{\textrm{(không thoả mãn)}}
\end{array} \right.$\\
TH2: $t = \dfrac{3}{2} \Leftrightarrow \sqrt {\dfrac{x^2 - 6x}{x + 3}}  = \dfrac{3}{2} \Rightarrow 4x^2 - 33x - 27 = 0 \Leftrightarrow \left[ \begin{array}{l}
x = 9\,\,{\textrm{(thoả mãn)}}\\
x =  - \dfrac{3}{4}{\textrm{(không thoả mãn)}}
\end{array} \right.$.\\
Vậy phương trình đã cho có 2 nghiệm là: $x = \dfrac{7 + \sqrt {61} }{2}$ và $x = 9$.\\

}
\end{ex}
